# SKIPA 서비스 개요

## 서비스 정보

| 항목 | 내용 |
|------|------|
| 서비스명 | SKIPA |
| 풀네임 | SK IP Agent |
| 성격 | SK AX 사내 특허 관리 시스템 |
| 목적 | Legal팀과 사업부가 협업하여 특허 유지/포기 의사결정을 효율적으로 수행 |

---

## 전체 서비스 구조

| 화면 구분 | 페이지 목록 |
|-----------|------------|
| 공통 화면 | 로그인 / 전체 특허 목록 / 특허 상세 / 만료 예정 특허 관리 |
| Legal팀 화면 | 홈 / 재평가 관리 / 포트폴리오 분석 / 만료 예정 특허 관리 |
| 사업부 화면 | 홈 / 검토 현황 / 담당 특허 관리 / 사전 평가 Lab |

---

## 재평가 프로세스 흐름

```
[Legal팀] 분기별 대상 특허 선정
    ↓
[Legal팀] 사업부 배정 및 검토 요청
    ↓
[사업부] 특허 상세 확인 + AI 평가 보고서 검토
    ↓
[사업부] 유지 / 포기 의견 제출
    ↓
[Legal팀] 회신 확인 및 최종 결정
```

### 특허 데이터 (20건 이상)
- 상태 혼합: 등록 / 만료 예정 / 포기+만료
- 국가 혼합: 한국 / 미국 / 일본 / 유럽 / 중국
- 사업부 혼합: 반도체사업부 / 통신사업부 / 에너지사업부 / 제조사업부
- 기술 분야: AI/ML / 반도체 / 통신 / 에너지 / 제조

### 분기 기준
- 현재 분기: 2025년 1분기
- 제출 마감일: 2025년 3월 31일

### 차트용 연도 데이터
- 범위: 2020 ~ 2025년
- 항목: 출원 / 등록 / 만료 건수
