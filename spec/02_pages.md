### 3. 특허 상세 `/patents/:id`

탭 구성:

| 탭 | 내용 |
|----|------|
| 특허 원문 | 특허번호, 명칭, 출원인, 발명자, 출원일, 등록일, 만료일, IPC, 요약, 청구항 |
| AI 평가 보고서 | 기술성 / 권리성 / 사업성 점수 + 종합 등급 (S/A/B/C) + AI 코멘트 |
| 유사 특허 분석 | 유사 특허 목록 테이블 (유사도 점수, 특허번호, 특허명) |
| 사내 프로젝트 연관 정보 | 연관 프로젝트 카드 목록 (프로젝트명, 담당부서, 연관도) |
| 유지/포기 의견 제출 | 유지 or 포기 라디오 선택 + 의견 텍스트 입력 + 제출 버튼 |

접근 권한:
- Legal팀: 모든 특허 접근 가능, 사업부 코멘트 함께 표시
- 사업부: 담당 특허만 접근 가능

### 5. Legal 홈 `/legal/home`

**재평가 파이프라인 (상단 전체 너비)**

| 구성요소 | 상세 |
|----------|------|
| 단계 표시 | 요청 전 → 요청 완료 → 지연 → 회신 완료 (화살표로 연결된 카드) |
| 각 단계별 건수 | 숫자로 표시 |
| 지연 항목 | 빨간색 강조, 기한 초과 건수 명시 |

**총 관리 특허 분포 (중단)**

| 구성요소 | 상세 |
|----------|------|
| 단계별 분포 | 등록 / 만료 예정 도넛 차트 |
| 국가별 분포 | 가로 막대 차트 |
| 기술 분야별 분포 | 바 차트 |
| 이번 분기 비용 요약 | 예상 연차료 금액 카드 |

**연간 연차료 지출 추이 (하단 좌측)**
- 연도별 연차료 라인 차트 (2020~2025)

**사업부 회신 알림 (하단 우측)**
- 최근 회신 목록: 사업부명, 특허번호, 특허명, 회신일시

### 12. 사전 평가 Lab `/dept/lab`

**레이아웃 구성**
- 좌측 패널: 이전 대화 목록 (세션별로 저장된 평가 이력)
- 우측 메인: 현재 평가 진행 영역

**입력 영역 (탭)**

| 탭 | 내용 |
|----|------|
| 직접 입력 | 특허명, 기술 설명, 출원 예정 국가, 관련 사업, IPC 등 필수 항목 폼 |
| PDF 업로드 | 드래그 앤 드롭 또는 파일 선택 |

- "평가 시작" Primary 버튼

**평가 보고서 영역**
- 평가 시작 후 결과 표시
- 기술성 / 권리성 / 사업성 각 점수 + 종합 등급 (S/A/B/C)
- 항목별 AI 코멘트

**챗봇 영역**
- 평가 결과 관련 질문 가능한 채팅 UI
- 대화 내역 저장 (좌측 목록에서 이전 대화 불러오기 가능)
- 입력창 하단 고정
